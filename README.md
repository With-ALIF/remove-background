# 🖼️ AI Background Remover 

![Vite](https://img.shields.io/badge/Vite-5.x-646CFF?logo=vite&logoColor=white)
![TypeScript](https://img.shields.io/badge/TypeScript-5.x-3178C6?logo=typescript&logoColor=white)
![remove.bg](https://img.shields.io/badge/API-remove.bg-4CAF50)
![License](https://img.shields.io/badge/License-Open--Source-brightgreen)
![Status](https://img.shields.io/badge/Status-Active-success)

A modern **frontend-only** web application built with **Vite + TypeScript** that allows users to:

- Upload **image**
- Remove backgrounds using **remove.bg API**
- Apply a **custom background color**
- Preview results
- Download edited images individually

---

🔗**Links:** **[🚀 Live Demo](https://remove-background-free.vercel.app/)** | **[🗄️ Server Repo](https://github.com/With-ALIF/remove-background)**
---

## 🚀 Features

- ✅ Multiple image upload
- ✅ Real AI background removal (remove.bg API)
- ✅ Custom background color picker
- ✅ Per-image preview (original + result)
- ✅ Individual download buttons
- ✅ Modern UI with SVG icons
- ✅ Built with Vite + TypeScript
- ✅ Netlify deploy ready

###  🧪 How It Works 
```
## 🧪 How It Works

1. User uploads one or multiple images
2. Images are sent to remove.bg API
3. Background is removed using AI
4. Canvas API is used to apply a custom background color
5. Final images are rendered and downloadable
```
